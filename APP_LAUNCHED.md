## âš ï¸ TROUBLESHOOTING

### Backend Won't Start
**Error:** "Port 8000 already in use"
**Solution:** Kill the process on port 8000:
```powershell
netstat -ano | findstr :8000
taskkill /PID <PID> /F
```

### Flutter Won't Compile
**Error:** "Failed to bind web development server on port 5002"
**Solution:** Try a different port:
```bash
flutter run -d chrome --web-port=5003
```

### Firebase Connection Error
**Error:** "Unable to connect to Firebase"
**Check:**
- Internet connection
- Firebase project ID (should be: asclepius-f664c)
- Firebase configuration in lib/firebase_options.dart

### Slow Performance
**Possible Causes:**
- Low disk space
- High CPU usage
- Network latency
- Browser cache

**Solutions:**
- Restart the application
- Clear browser cache
- Close unnecessary programs
- Check internet speed
